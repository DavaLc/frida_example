# frida_example

### 基础函数

- python rpc: index.py
- byte[]转字符串： byte2str.js
- 枚举所有的类：list_class.js
- 枚举类中的方法：list_func_from_class.js
- 打印堆栈：printstack.js

### 进阶脚本
- 跟踪某个class或者function，并且打印出参数和返回值：trace_class.js
- hook常见的加解密方法：crypt.js
